## Usando o GIT
Primeiro crie uma pasta, e acesse ela usando o terminal

```
// dir = exibi o diretório
dir

// cd = navega entre as páginas 
cd Desktop

// mkdir = cria a pasta
mkdir pastaTeste

cd pastaTeste
```
